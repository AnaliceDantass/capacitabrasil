# Capacita Brasil

Esse repositório foi feito com o objetivo de armazenar minhas atividades práticas voltadas ao Programa Capacita Brasil, programa o qual pude participar visando aprimorar os meus conhecimentos técnicos acerca da área de Tecnoloia da Informação e Comunicação.
Ao transcorrer da trilha de conhecimento que escolhi, a qual foi Provimentoo de Serviços computacionais, pude recordar de conceitos básicos como a lógica de programação, e consolidar meus conhecimentos acerca do desenvolvimento em nuvem.
